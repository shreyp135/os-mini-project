## Running the Shell

### Compile:
```bash
gcc basic_shell.c -o basic_shell
```
### Run 
```bash
./basic_shell
```

## Example commands

Once the shell starts, you can run any Linux command, for example:

### List files

```bash
myshell> ls
```

### Print working directory
```bash

myshell> pwd
```

### Print text
```bash

myshell> echo hello
```

### Exit
```bash

myshell> exit
```
